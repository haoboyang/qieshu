# qieshu 识别内容后台管理系统 sdk
简介
此 SDK 适用于 Java 8版本。使用此 SDK 构建您自己识别内容后台管理系统能让您以非常便捷地方式管理您的箧书识别设备及识别内容。

Java SDK 属于箧书SDK之一，主要有如下功能：

1.提供识别组操作管理,识别组授权
2.提供识别 app 管理
3.提供专业超声波设备管理
4.提供设备内容识别设备管理
5.提供第三方设备动态音频获取

#安装
<dependency>
			<groupId>net.qieshu</groupId>
			<artifactId>qieshu-sdk</artifactId>
			<version>2.0</version>
		</dependency>
